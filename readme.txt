Displays list of posts of the current authorized user

== Description ==
This simple widget will be useful primarily for multi-user sites, the content of which is created and edited by users from the Front End. For example: web-boards about purchases, sales, rental, lost and found, services, etc.

Plugin features:

* Display only for authorized users
* Settings of the number of posts in the list
* Settings showing / hiding the dates of posts
* Support for any custom [shortcodes]


== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/ list-of-user-posts` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the \'Plugins\' screen in WordPress.
1. Use the Appearance->Widgets->List of User Posts to configure the widget.
